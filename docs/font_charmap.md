Notes
-----
- $02, $03, $0F are all identical full blocks in the BIOS data
- $81-$85 and $A1-$A5 share the same punctuation characters
- $90/$B0 share the prolonged sound mark
- $FE/$DE share dakuten, $FF/$DF share handakuten
- $80-$9F + $E0-$FF = hiragana (mirrors $A0-$DF katakana layout)
- $A1-$DF matches JIS X 0201 half-width katakana exactly
- $1C-$1F are SNK logo tiles with no Unicode equivalent; substitute
  font should provide placeholder graphics or leave empty
- $04-$06 are custom UI elements (vertical bar, horizontal bar, corner)
  used by the BIOS for progress/selection display
